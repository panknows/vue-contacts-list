# vue-contacts-list

[Live demo](https://panknows.github.io/vue-contacts-list)

## Features

- Add, edit & remove contact;
- Search by name;

## Dependencies

- Vue 2.6.14;
- Vue-router 3.5.1;
- Vuex 3.6.2;
- Bootstrap 4.6.1;
- Bootstrap-vue 2.21.2;

## Project setup
```
npm install
```
